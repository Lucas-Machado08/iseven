# Dev Resources

Esta pasta contém arquivos de desenvolvimento, templates e ferramentas de teste.

## Arquivos

- **email-template.html** - Template HTML para configurar no EmailJS
- **gerar-imagem-teste.html** - Ferramenta para gerar imagens de teste de contas de energia

## Como usar

### Email Template
1. Abra `email-template.html`
2. Copie o código HTML
3. Cole no painel do EmailJS em Email Templates

### Gerador de Imagem de Teste
1. Abra `gerar-imagem-teste.html` no navegador
2. Clique em "Baixar Imagem de Teste"
3. Use a imagem gerada para testar o formulário
